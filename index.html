Cloud Server là gì?

Cloud Server là máy chủ ảo được xây dựng dựa trên nền tảng công nghệ điện toán đám mây (Cloud).

Cloud Server là cuộc cách mạng mới cho giải pháp lưu trữ website, nhằm thay thế và khắc phục hoàn toàn những nhược điểm của máy chủ truyền thống.

Đăng ký ngay

Giao diện Control Panel thân thiện với người dùng

Phát triển bởi đội lập trình với hơn 100 kỹ sư hệ thống trong 2 năm.

Giao diện quản trị được thiết kế đơn giản, tiện dụng.

Thiết lập DB và Mail Server chỉ với 1 click, sử dụng ngay lập tức.

Cho phép khách hàng quản lý và phân bổ tài nguyên cho các máy chủ ảo.

Không bị downtime khi Server gặp sự cố

Hệ thống Cloud đạt chuẩn quốc tế, thiết kế bởi đội ngũ chuyên nghiệp từ Nhật Bản.

Cơ chế tự theo dõi trạng thái của máy chủ trên Cloud Server, tự động chuyển đổi dữ liệu giữa các máy chủ khi xảy ra sự cố.

Hệ thống vận hành liên tục với thời gian uptime lên đến 99.9%.

Server DELL R630 và đường truyền tốc độ 10 Gbps

Tất cả máy chủ đều là DELL R630 với hiệu năng tốt nhất.

Các thiết bị mạng có tốc độ kết nối 10 Gbps để tăng tốc độ truyền tải và sao lưu dữ liệu. (Máy chủ thông thường tốc độ chỉ đạt 1 Gbps)

Tốc độ xử lý vượt trội

Ổ cứng SSD Enterprise giúp tăng tốc độ truy xuất dữ liệu.

Nhanh hơn gấp 5 lần ổ cứng SSD thông thường.

Hệ thống lưu trữ dữ liệu tập trung SAN (Storage Area Network) tăng sự ổn định và khả năng nâng cấp dung lượng lưu trữ.

Sao lưu và khôi phục hình ảnh qua Control Panel

Khách hàng tự quản lý sao lưu và khôi phục lại hình ảnh thông qua Control Panel.

Chức năng tự động sao lưu Auto Backup.

Tự điều chỉnh dung lượng máy chủ

Thêm máy chủ ảo trên Cloud Server một cách dễ dàng với chỉ 1 cú click.

Tự điều chỉnh các thông số: CPU, Bộ nhớ, Dung lượng ổ cứng qua Control Panel.

Cloud Server dùng được cho các ứng dụng phổ biến

Nguồn mở CMS: Wordpress, Drupal, Joomla ...

Trang web thương mại điện tử - Magento, WooCommerce, PrestaShop, osCommerce hoặc ZenCart.

Các trang web hình ảnh độ nét cao lưu nhiều danh mục khách hàng.

Các trang web nặng về video.

Diễn đàn thảo luận.

Trang web cộng đồng.

Trang web lưu trữ các tài liệu nặng.

Danh sách doanh nghiệp hoặc dịch vụ.

Trang web bán đấu giá.

Ứng dụng phương tiện truyền thông xã hội.

Tích hợp rất nhiều hệ điều hành

Có thể lựa chọn hệ điều hành Linux và Windows phù hợp với trang web của bạn.

Đối với hệ điều hành Linux, có 3 lựa chọn: CentOS (phổ biến nhất), Ubantu và Debian.

Hệ điều hành Windows với phiên bản Windows2012 mới nhất, 100% bản quyền.
